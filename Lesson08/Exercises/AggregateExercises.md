# Aggregate Exercises

## Exercise 1: Basic Product

Using the `Aggregate` method, calculate the product of numbers from 1 to 5. Print the result and the breakdown of the calculation.

## Exercise 2: Building a Dictionary

Create a dictionary where the keys are numbers from 1 to 5 and the values are their squares. Use `Aggregate` to build this dictionary and print the result.

## Exercise 3: Accumulating State with List

Count how many iterations it takes to sum the numbers from 1 to 10, and store each intermediate sum in a list. Print the final count and the list of sums.

## Exercise 4: Nested Aggregate with Lists

For each number from 1 to 3, create a list of strings that represent the multiplication of that number with numbers from 1 to 2. Print all results.

## Exercise 5: Nested Aggregate with State

Simulate a game where three players take turns drawing cards from a deck of 10 cards (represented as a `List<int>` from 1 to 10). Each player should draw one card per round for three rounds. Print the final hands of each player.

## Exercise 6: Card Dealing Pattern

Implement a card dealing pattern where you deal 5 cards to each of 4 players from a deck of 52 cards (represented as a `List<int>` from 1 to 52). Print the remaining cards in the deck and the hands of each player.
