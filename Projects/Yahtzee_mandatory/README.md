This is a fully functioning 13 round Yahtzee Simulator built with functional programing patterns in C#. 

Requirements are net10.0

How to Run
**1.Clone the repository**
Use the command in your console:
git clone https://github.com/SweKingdom/YahzeeSimulationProject.git

**2. Run the project**

Use the command in your console in your IDE:
dotnet run

The simulation will print 13 rounds of gameplay and a final score card for all the players.
